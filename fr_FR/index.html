Le Web Design en 4 minutes

par Jérémy Thomas

Disons que vous avez un produit, un portfolio ou simplement une idée que vous souhaitez partager avec tout le monde sur votre propre site Web. Avant de le publier sur Internet, vous voulez le rendre attrayant, professionnel ou au moins décent à regarder.

Quelle est la première chose sur laquelle vous devez travailler ?

Le contenu

Le but du design est d'améliorer la présentation du contenu auquel il est appliqué. Cela peut sembler évident, mais le contenu étant l'élément principal d'un site Web, il ne doit pas être établi après coup.

Le contenu écrit, comme le paragraphe que vous lisez actuellement, représente plus de 90 % du Web. Donner un style à ce contenu est une étape importante.

Supposons que vous ayez déjà finalisé le contenu que vous souhaitez publier et que vous veniez de créer un fichier style.css vide, quelle est la première règle que vous pouvez écrire ?

Centrer les éléments

Les longues lignes de texte peuvent être difficiles à parcourir, et donc difficiles à lire. Le choix d'une limite de caractères par ligne améliore considérablement la lisibilité et l'attrait d'un mur de texte.

body {
  margin: 0 auto;
  max-width: 50em;
}

Après avoir stylisé les blocs de texte, pourquoi ne pas styliser le texte lui-même?

Famille de police

La police du navigateur par défaut est "Times", ce qui peut sembler peu attrayant (principalement parce qu’il s’agit de la police par défaut). Passer à une police sans empattement comme "Helvetica" ou "Arial" peut considérablement améliorer l’apparence de votre page.

body {
  font-family: "Helvetica", "Arial", sans-serif;
}

Si vous voulez vous en tenir à une police serif, essayez "Georgia".

Bien que cela rende le texte plus attrayant, rendons-le également plus lisible.

Espacement

Lorsqu'une page semble « cassée » pour un utilisateur, il s'agit généralement d'un problème d' espacement . Fournir de l'espace à la fois autour et dans votre contenu peut augmenter l'attrait de votre page.

body {
  line-height: 1.5;
  padding: 4em 1em;
}

h2 {
  margin-top: 1em;
  padding-top: 1em;
}

Bien que la mise en page se soit considérablement améliorée jusqu'à présent, appliquons des modifications plus subtiles.

Couleur & contraste

Un texte noir sur fond blanc peut être agressif pour les yeux. Opter pour une nuance de noir plus douce pour le corps du texte rend la page plus confortable à lire.

body {
  color: #555;
}

Et afin de garder un bon niveau de contraste, choisissons une teinte plus foncée pour les mots importants

h1,
h2,
strong {
  color: #333;
}

Bien que la majeure partie de la page ait été améliorée visuellement, certains éléments (comme les extraits de code) semblent encore inappropriés.

Equilibre

Il suffit de quelques touches supplémentaires pour corriger l'équilibre de la page :

code,
pre {
  background: #eee;
}

code {
  padding: 2px 4px;
  vertical-align: text-bottom;
}

pre {
  padding: 1em;
}

À ce stade, vous voudrez peut-être faire ressortir votre page et lui donner une identité.

Couleur d'accentuation

La plupart des marques ont une couleur qui agit comme un accent visuel. Sur un site Web, cette couleur peut être utilisé pour mettre l'accent sur des éléments interactifs, comme des liens.

a {
  color: #e81c4f;
}

Mais pour garder l'équilibre, nous aurons besoin de quelques couleurs complémentaires.

Couleur complémentaire

La couleur d'accentuation peut être complétée par des nuances plus subtiles , à utiliser sur les bordures, les arrière-plans ou même le corps du texte.

body {
  color: #566b78;
}

code,
pre {
  background: #f5f7f9;
  border-bottom: 1px solid #d8dee9;
  color: #a7adba;
}

pre {
  border-left: 2px solid #69c;
}

Après avoir changé les teintes, pourquoi ne pas changer les formes...

Police personnalisée

Le texte étant le contenu principal d'une page Web, l'utilisation d'une police personnalisée confère à la page une identité encore plus personnelle.

Vous pouvez importer votre police personnalisée, ou utiliser un service en ligne comme Typekit. Ici nous utiliserons la police "Roboto" mise à disposition par le service gratuit Google Fonts:

@import 'https://fonts.googleapis.com/css?family=Roboto:300,400,500';

body {
  font-family: "Roboto", "Helvetica", "Arial", sans-serif;
}

Après avoir amélioré votre identité visuelle par le style du texte, pourquoi ne pas ajouter un millier de mots...

Images et icônes peuvent être utilisés, soit comme décoration de votre contenu, soit prendre activement part au message que vous souhaitez faire passer.

Améliorons notre en-tête avec une belle image de fond du site Unsplash

header {
  background-color: #263d36;
  background-image: url("header.jpg");
  background-position: center top;
  background-repeat: no-repeat;
  background-size: cover;
  line-height: 1.2;
  padding: 10vw 2em;
  text-align: center;
}

Ajoutons également un logo

header img {
  display: inline-block;
  height: 120px;
  vertical-align: top;
  width: 120px;
}

Profitons également de l'opportunité pour améliorer le style de nos textes.

header h1 {
  color: white;
  font-size: 2.5em;
  font-weight: 300;
}

header a {
  border: 1px solid #e81c4f;
  border-radius: 290486px;
  color: white;
  font-size: 0.6em;
  letter-spacing: 0.2em;
  padding: 1em 2em;
  text-transform: uppercase;
  text-decoration: none;
  transition: none 200ms ease-out;
  transition-property: color, background;
}

header a:hover {
  background:  #e81c4f;
  color: white;
}

Et voilà!

Nous avons créé une page décente en quelques minutes seulement, suivant des principes élémentaires de Web Design. Il ne reste qu'une une seule chose à faire...
