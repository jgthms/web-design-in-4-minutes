Diseño Web en 4 minutos

por Jeremy Thomas

Vamos a decir que tienes un producto, un portafolio, o solo una idea que quieres compartir con todos en tu propia página web. Antes de publicarla en internet, quieres hacerla atractiva, profesional, o al menos decente de ver.

¿Cuál es la primera cosa en la que necesitas trabajar?

Contenido

El proposito del diseño es mejorar la presentación del contenido al que es aplicado. Puede sonar obvio, pero el contenido es el elemento principal de una página web, no debe ser definido en el último momento.

Luego de escribir el contenido, como el parrafo que estás leyendo ahora, tienes el 90% de la página web hecha. Para poner estilos a este contenido es necesario pasar por un largo camino.

Vamos a asumir que ya has finalizado el contenido que quieres publicar y que acabas de crear un archivo vacío style.css, ¿cuáles son las primeras líneas que debes escribir?

Centrar

Líneas de texto largas pueden ser difíciles de dividir, y en concecuencia difíciles de leer. Configurar un límite de caracteres por línea mejorar de gran manera la legibilidad y el atractivo de un bloque de texto. 

body{
  margin: 0 auto;
  max-width: 50em;
}

Después de estilizar los bloques de texto, ¿qué pasa con el estilo del texto?

Familia tipográfica

La tipografía por defecto de los navegadores web es "Times", que puede parecer poco atractiva (la mayoría de las veces porque es una tipografía "sin estilo"). Cambiando a tipografías sans-serif como "Helvetica" o "Arial" puede mejorar de gran manera la apariencia de tu página web. 

body {
  font-family: "Helvetica", "Arial", sans-serif;
}

Si quieres mantener una tipografía serif, prueba con "Georgia".

Mientras esto hace el texto más atractivo, vamos también hacerlo más legible.

Espacios

Cuando una página web luce como si "algo le faltara" a un usuario, es usualmente por problemas de espacios. Proporcionar espacios fuera y dentro de tu contenido puede incrementar el atractivo de tu página web. 

body {
  line-height: 1.5;
  padding: 4em 1em;
}

h2 {
  margin-top: 1em;
  padding-top: 1em;
}

Aprovechando que el diseño ha mejorado muchísimo, vamos a aplicar más cambios sutiles.

Color & contraste

El texto negro en un fondo blanco puede ser demasiado fuerte. Optar por una tonalidad más clara del negro para el texto hace la página web más cómoda de leer. 

body{
  color: #555;
}

Y con el fin de mantener un nivel decente de contraste, vamos a escoger una tonalidad más oscura para palabras importantes. 

h1,
h2,
strong {
  color: #333;
}

Aunque la mayoría de la página web ha mejorado visualmente, algunos elementos (como los bloques de código) aún parecen fuera de lugar.

Balance

Solo necesitamos unos pequeños retoques para corregir el balance de la página web: 

code,
pre{
  background: #eee;
}

code {
  padding: 2px 4px;
  vertical-align: text-bottom;
}

pre {
  padding: 1em;
}

En este punto, tal vez quieres destacar tu página web y darle identidad.

Color principal

Muchas marcas tiene un color primario que sirve para resaltar visualmente. En una página web, este realce puede ser usado para dar enfasis en elemento interactivos, como enlaces. 

a {
  color: #e81c4f;
}

Pero para mantener el balance, necesitaremos algunos colores adicionales.

Colores secundarios

El realce de color puede ser complementado con tonalidades más sutiles, usandos en bordes, fondos, o incluso el cuerpo del texto. 

body {
  color: #566b78;
}

code,
pre {
  background: #f5f7f9;
  border-bottom: 1px solid #d8dee9;
  color: #a7adba;
}

pre {
  border-left: 2px solid #69c;
}

Luego de cambiar las tonalidades, ¿por qué no cambiar la forma...?

Tipografía personalizada

Como el texto es el contenido principal de la página web, el uso de una tipografía personalizada da una identidad incluso más perceptible.

Aunque puedes incluir tu propia tipografía o usar un servicio en línea como Typekit, vamos a usar "Roboto" del servicio gratuito Google Fonts: 

@import 'https://fonts.googleapis.com/css?family=Roboto:300,400,500';

body {
  font-family: "Roboto", "Helvetica", "Arial", sans-serif;
}

Después de mejorar la identidad de la página web usando texto, ¿qué pasa si agregamos unas mil palabras ...?

Gráficos e iconos pueden ser usados para apoyar tu contenido, o para tomar parte activa en el mensaje que quieres transmitir.

Vamos a mejorar nuestro encabezado con una bonita imagen de fondo de Unsplash 

header {
  background-color: #263d36;
  background-image: url("header.jpg");
  background-position: center top;
  background-repeat: no-repeat;
  background-size: cover;
  line-height: 1.2;
  padding: 10vw 2em;
  text-align: center;
}

Vamos a agregar un logo

header img {
  display: inline-block;
  height: 120px;
  vertical-align: top;
  width: 120px;
}

Vamos a aprovechar esta oportunidad para mejorar los estilos de los textos. 

header h1 {
  color: white;
  font-size: 2.5em;
  font-weight: 300;
}

header a {
  border: 1px solid #e81c4f;
  border-radius: 290486px;
  color: white;
  font-size: 0.6em;
  letter-spacing: 0.2em;
  padding: 1em 2em;
  text-transform: uppercase;
  text-decoration: none;
  transition: none 200ms ease-out;
  transition-property: color, background;
}

header a:hover {
  background:  #e81c4f;
  color: white;
}

¡voilà!

Hemos diseñado una página web decente en solo unos minutos, siguiendo estos principios básicos de diseño web. Solo hay una última cosa por hacer...
