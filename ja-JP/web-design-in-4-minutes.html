4分でウェブデザイン

by Jeremy Thomas

例えばあなたが作った製品、ポートフォリオ、またはただのアイデアを自分の ウェブサイトにのせてみんなと共有したいと思ったとしましょう。インターネットに公開する前に魅力的・プロっぽい、少なくともまともな見た目にしたいですよね。

そのために 最初に しなくてはいけないことは何でしょう？

コンテンツ

デザインの目的は、そのデザインをあてるコンテンツの表現力を増すことです。当然のように聞こえるでしょうが、コンテンツがウェブサイトで一番重要な要素であって、決してデザインの付けたしであってはいけません。

ウェブの90%以上を占めるのは、あなたが今読んでいるような文章のコンテンツです。ですから、文章のスタイルを整えることができれば十分効果があります。

あなたが既に公開したいコンテンツは書き終えて、まだなにも書かれていないstyle.css ファイルを作ったばかりだとしましょう。 あなたが書くべき最初のルール は何でしょう？

中央寄せ

1行あたりの文字数が多いと解析するのが難しく、従って読むのが大変です。　1行あたりの文字数を制限することで、格段によみやすくなり、素敵な文字の壁ができます。

body {
  margin: 0 auto;
  max-width: 50em;
}

文字の かたまりのスタイル整えることができたら、文字そのもののスタイルを整えるのはどうでしょう？

フォント

ブラウザの標準のフォントは"Times"で、これは魅力的には見えません（"スタイル"が整っていないフォントだからでしょう）。 サンセリフ体 のフォント、例えば "Helvetica" または "Arial" に変更すると、あなたのページを大幅に改善できます。

body {
  font-family: "Helvetica", "Arial", sans-serif;
}

もしセリフ体にこだわりがあるののなら "Georgia"を試してみてください。

これにより、文字が魅力的になりました。さあ、それではより読みやすくしてみましょう。

余白

ページが "壊れている" ように見える理由は、たいてい 余白 の問題です。　余白をコンテンツの周り と 内側に設定すると、ページが魅力的になります。

body {
  line-height: 1.5;
  padding: 4em 1em;
}

h2 {
  margin-top: 1em;
  padding-top: 1em;
}

レイアウトが大幅に改善されましたので、次は 微妙な変化をさせましょう。

色 & コントラスト

白地に黒の文字色は目につらい場合があります。文字色を落ち着いた色合いにすることで、ページを読むのがより快適 になります。

body {
  color: #555;
}

そしてコントラストを適切にするため、重要な 言葉の文字色には濃い色合いを選びましょう。

h1,
h2,
strong {
  color: #333;
}

ページのほとんどの部分の見た目は改善されましたが、いくつかの要素(例えばプログラムコード) は依然として適切ではありません。

バランス

ページのバランスを整える為、ちょっとだけ変更します:

code,
pre {
  background: #eee;
}

code {
  padding: 2px 4px;
  vertical-align: text-bottom;
}

pre {
  padding: 1em;
}

この時点で、あなたはページを個性的にして目立つようにしたくなっていることでしょう。

第一の色

ほとんどのブランドには第一の色があり、見た目上のアクセントになっています。ウェブサイトでは、このアクセントはリンクといったユーザが操作できる要素を強調するために使用します。

a {
  color: #e81c4f;
}

バランスを保つために、いくつか 追加の色が必要になります。

第二の色

アクセントの色は、薄い 色あいによってより引き立ちます。ボーダーや背景色、本文の文字色にも使うことができます。

body {
  color: #566b78;
}

code,
pre {
  background: #f5f7f9;
  border-bottom: 1px solid #d8dee9;
  color: #a7adba;
}

pre {
  border-left: 2px solid #69c;
}

色合いをかえるのであれば、形を変えない理由はないですよね...

カスタムフォント

ウェブページでは本文が大事なのですから、カスタムフォントを使うとページがより目立ち、独自のものになります。

独自のウェブフォントやTypekitのようなオンラインサービス使うことができます。それでは、無料で使うことができるGoogle Fonts のサービスから"Roboto" を使ってみましょう。

@import 'https://fonts.googleapis.com/css?family=Roboto:300,400,500';

body {
  font-family: "Roboto", "Helvetica", "Arial", sans-serif;
}

言葉で独自性を発揮することができたら、1000の言葉に匹敵するものを追加するのはどうでしょう...

画像とアイコンはあなたのコンテンツを飾ったり、あなたのメッセージを伝えるのに一役買います。

それではUnsplashの素敵な背景画像 を使って、ヘッダをより良くしてみましょう。

header {
  background-color: #263d36;
  background-image: url("header.jpg");
  background-position: center top;
  background-repeat: no-repeat;
  background-size: cover;
  line-height: 1.2;
  padding: 10vw 2em;
  text-align: center;
}

ロゴも追加しましょう。

header img {
  display: inline-block;
  height: 120px;
  vertical-align: top;
  width: 120px;
}

文字のスタイルを強化してみましょう。

header h1 {
  color: white;
  font-size: 2.5em;
  font-weight: 300;
}

header a {
  border: 1px solid #e81c4f;
  border-radius: 290486px;
  color: white;
  font-size: 0.6em;
  letter-spacing: 0.2em;
  padding: 1em 2em;
  text-transform: uppercase;
  text-decoration: none;
  transition: none 200ms ease-out;
  transition-property: color, background;
}

header a:hover {
  background:  #e81c4f;
  color: white;
}

じゃじゃーん!

わたしたちはほんの数分で、ウェブデザインの基本原則に従ってウェブページをまともなデザインにしてきました。あともう一つだけやることがあります...

ラブを共有しよう!
